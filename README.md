# Osoy
Script manager for interpreted programs, including, but not limited to, python programs, nodejs programs \& shell scripts.

## File structure
    ~/.osoy/
      bin/
        <symlink> -> <executable>
        ...
      packages/
        <domain>/
          ...
          <author>/
            ...
          <author>/
            <package>/
              ...
            <package>/
              <modules>/
              <module>
              <executable>
              ...
